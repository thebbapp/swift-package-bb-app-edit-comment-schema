# ``BbAppEditCommentSchema``

Data structures for comment editing.

## Overview

The BbAppEditCommentSchema library defines the data structures and schemas used in comment editing workflows. It provides type definitions for edit comment requests, response data, validation rules, and the data models that coordinate between the edit comment UI and the underlying content sources.
